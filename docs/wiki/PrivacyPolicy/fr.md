# Règles de confidentialité de Librera Reader


# # 1. Informations personnelles

Nous ne collectons aucune information personnelle des utilisateurs de nos applications.
Google Wallet nous permet de voir un minimum de détails de facturation pour chaque vente - le nom sur la carte et une adresse de facturation.
Google Wallet conserve une trace de ces informations, mais nous n'enregistrons, stockons, suivons ou divulguons ces informations à qui que ce soit.

Si vous nous envoyez une requête ou des commentaires par e-mail, votre adresse e-mail sera visible pour nous et si une réponse est nécessaire, nous utiliserons votre adresse e-mail pour vous contacter directement.

# # 2. Divulgation d'informations personnelles

Nous ne divulguons pas vos informations personnelles à des tiers.

# # 3. Autorisations requises

L'application nécessite différentes autorisations en fonction de leurs fonctionnalités:

La permission &quot;WRITE_EXTERNAL_STORAGE&quot; est utilisée pour permettre au lecteur de sauvegarder des fichiers de livre et des caches d’informations de livre;
vous pouvez également gérer vos fichiers enregistrés sur le stockage local avec le lecteur directement avec cette autorisation.

# # 4. Licence limitée

Librera Reader vous accorde une licence révocable limitée, non exclusive, non transférable pour utiliser l'application.
Vous ne pouvez utiliser l'application que sur les appareils que vous possédez ou contrôlez, ainsi que le permettent les conditions de service de la boutique d'applications sur lesquelles vous avez peut-être téléchargé l'application.
