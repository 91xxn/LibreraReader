# 如何将PDF转换为EPUB

> Librera可以使用在线服务将任何格式转换为任何格式，或使用文本重排模式（移动模式）将PDF转换为EPUB

### 在线转换器

在左侧菜单中找到@ #Online文件转换器@＃

选择“转换为EPUB”

* 将PDF转换为EPUB
* 选择文档并启动流程
* 下载EPUB Book
