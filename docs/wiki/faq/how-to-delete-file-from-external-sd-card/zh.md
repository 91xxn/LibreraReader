{1}如何从外部SD卡中删除书籍

Android 7+不允许通过外部SD卡删除文件
安全原因。
您需要添加SD卡作为资源，并允许应用程序从中删除文件

{4}打开文件管理器选项卡
{4}单击“添加资源”
{4}选择SD卡
{4}选择并允许从选择的SD卡中删除
